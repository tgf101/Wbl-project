# Church Website Chatbot System

# Project Overview 
This project focuses on the design and development of a chatbot integrated into a church website to imrpove communication, accessibility, and user engagement. The chatbot enables members and visitors to quickly access information such as service times, upcoming events, and contact details. In addition a discussions page was added using wordpress to allow users to interact with one another, share ideas, and ask questions. 

# Technologies used 
HTML, CSS, JavaScript, WordPress, Google Gemini API.

# Features 
Chatbot interface with real time responses
Typing indicator for user experience 
AI generated responses for church related questions 
community discussions page with comment functionality.

# System Architecture 

The system has a client-side architecture where the chatbot works within the browser. User input is processed using JavaScript which is sent to the Google Gemini API that then generates a response and returns it to the chatbot interface. 

# How it works
User open the chatbot toggle interface to open the chatbot
User enters a query
JavaScript then processes this input which is sent to the Google Gemini API
Response is then returned and displayed
